# 功能描述
基于siamese-lstm的中文句子相似度计算

# 环境搭建
* Ubuntu：16.04（64bit）
* Anaconda：2-4.4.0（python 2.7）

历史版本下载：<https://repo.continuum.io/archive/>
* TensorFlow：1.2.1
* numpy：1.14.2
* gensim：3.4.0
* nltk：3.2.3
* jieba：0.39
* word2wec中文训练模型

参考链接：<https://weibo.com/p/23041816d74e01f0102x77v>

# 代码使用

### 模型训练
        # python train.py

### 模型评估
        # python eval.py
